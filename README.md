# Projeto-Produtos
Sistema de Gerenciamento de Produtos em Python

Este projeto consiste em um sistema de gerenciamento de produtos em linha de comando (CLI), desenvolvido em Python, com foco em organização, boas práticas e manipulação de dados.
O sistema permite listar, cadastrar, alterar e remover produtos, simulando um controle básico de estoque.
Ele foi estruturado utilizando programação orientada a objetos (POO), onde a classe Produto é responsável por encapsular os dados e as operações relacionadas aos produtos.

O objetivo principal é oferecer um CRUD completo (Create, Read, Update, Delete) de produtos, permitindo ao usuário:
- Visualizar todos os produtos cadastrados
- Inserir novos produtos
- Alterar produtos existentes
- Remover produtos
- Interagir de forma simples via terminal

Tecnologias Utilizadas

- Python 3
- Programação Orientada a Objetos (POO)
- Estrutura de menus com match/case
- Tratamento de exceções (try/except)
- Interface em linha de comando (CLI)
